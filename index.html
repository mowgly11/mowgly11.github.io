Remember School

RememberSchool is a software that provides useful tools for students in school. Our main focus is helping students build the perfect portfolio for their future endeavors such as college, trade school, military school, ect.

it was built with NodeJS and Express for http requests handling, Html, css, Javascript and webpack as UI technologies, and mongodb as a database including some very popular libraries.

This project serves as a valuable tool for anyone in need of school assistance and advice on improving their studying experience, and is a great addition to any portfolio.

PasteBlend

PasteBlend is an easy to use tool for transfering text or files between 2 devices using a custom codes system.

Pasteblend was built using NodeJS express and socket.io as backend technologies, and EJS, Css, Javascript and webpack for the frontend.

Pasteblend is a valuable tool to allow users share their files between their devices without having to download or authenticate into any service as its fast, reliable and secure.

Earth Bot

Earth bot is a powerful discord bot focused mainly in making your discord server fun with its leveling and economy system, as well as social activities and games.

Earth bot was built using NodeJS and the Discord.js library to facilitate interacting with the discord api. currently, Earth bot is located in more than 1400 discord servers and many people are using it daily.

This project serves as a valuable tool for anyone interested Making their Discord server active and Fun. It's also a great addition to any portfolio as it showcases my ability to create and interact with external API's and technologies.
