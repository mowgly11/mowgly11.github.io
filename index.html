OTools API

Otools API is a powerful API that helps you with a bunch of online useful tools that can be used in any projects. it was built with NodeJS, Express and Passport.js including some very popular libraries like bcrypt, mongoose, sendgrid...

Otools API is using one of the fastest cloud computing services and high quality code which makes it fast enough to use in production.

This project serves as a valuable tool for anyone in need of quick and reliable online tools API, and is a great addition to any portfolio.

And what makes Earth Bot even more special is the speed and The high optimisation which makes it able to serve multiple requests without any slow or crash errors.

This project serves as a valuable tool for anyone interested Making their Discord server active and Fun. It's also a great addition to any portfolio as it showcases my ability to create and interact with external API's and technologies.
